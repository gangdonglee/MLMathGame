# 7살 수학게임 — 컨셉 문서

## 디자인 원칙

1. **글자 최소화** — 7살은 한글 읽기 부담이 있음. 아이콘·색·소리로 의도를 전달
2. **큰 버튼·큰 글씨** — 모바일/태블릿 터치 친화 (최소 80px 사각 영역)
3. **즉시 피드백** — 정답=색종이/별/소리, 오답=흔들림(부드럽게, 야단 안 침)
4. **실패 페널티 없음** — 틀려도 다시 시도. "실수하면 더 멋있어"
5. **파스텔 색감** — 채도 낮은 따뜻한 색. 눈 안 아프게
6. **한 게임 1~3분** — 집중력 한계 고려

## 미니게임 7종

### 기초 영역 (A)

#### 1. 숫자 풍선 터뜨리기
- **룰**: 화면 위쪽에 식이 뜸(`3 + 4 = ?`). 아래에서 풍선 4개가 떠오르고, 각각 숫자가 적혀 있음. 정답 풍선을 터뜨림
- **난이도**: 1~10 사이 덧셈/뺄셈
- **수학 직관**: 수와 연산 — 가장 기본

#### 2. 도형 친구 찾기
- **룰**: 가운데에 큰 도형(예: 빨간 동그라미). 주변에 6개 도형. 같은 모양인 것 모두 클릭(색·크기는 달라도 됨)
- **난이도**: 동그라미/세모/네모/별 4종
- **수학 직관**: 분류 — 본질적 속성과 우연적 속성 구분

#### 3. 다음에 올 건?
- **룰**: 패턴 줄(`🔴🔵🔴🔵🔴?`). 아래 보기 3개 중 다음에 올 것 고르기
- **난이도**: ABAB → ABCABC → AABBAABB
- **수학 직관**: 패턴/규칙 인식 — 함수 사고의 씨앗

### ML 번역 영역 (B)

#### 4. 별 모으기 모험 (벡터)
- **룰**: 격자판에 별 3개. 화살표 버튼(↑↓←→) 눌러 캐릭터를 움직여 별 모두 모으기. 화살표 1번 = 1칸
- **변형**: "위로 2칸 + 오른쪽 3칸" 같은 누적 명령
- **수학 직관**: 위치 = 시작 + 방향 합 (벡터 덧셈의 직관)
- **출처**: [01_선형대수1_벡터.md](../ML_수학_교과서/01_선형대수1_벡터.md)

#### 5. 공을 골짜기로 (그래디언트) — **v2: 기울이기 물리**
- **룰**: 울퉁불퉁한 언덕에 공. **화면을 좌우로 드래그하면 세상이 기울어지고**, 공이 진짜 물리 (중력+마찰+경사) 로 굴러감. 가장 큰 별이 있는 골짜기에 정착시키면 클리어
- **인터랙션 동사**: 끌기/기울이기 (탭 X)
- **수학 직관**: 자기가 곧 그래디언트 — "어느 쪽이 내리막"을 자기가 결정. 물리 관성으로 얕은 골짜기를 빠져나와 더 깊은 곳으로 가야 함 = local min 탈출의 직관
- **라운드**: 1 골짜기 → 2 골짜기 (얕은 거 빠져나오기) → 3 골짜기 (가장 깊은 거 분별)
- **출처**: [04_미적분1_그래디언트.md](../ML_수학_교과서/04_미적분1_그래디언트.md), [09_최적화1_볼록성_GD.md](../ML_수학_교과서/09_최적화1_볼록성_GD.md), [10_최적화2_Momentum_Adam_ES.md](../ML_수학_교과서/10_최적화2_Momentum_Adam_ES.md)

#### 6. 사탕 뽑기 통 (확률)
- **룰**: 통 두 개. 각 통에서 10번 사탕 뽑아보기 → 어느 통이 빨간 사탕 더 많이 나왔는지 맞추기. 통 안은 안 보임
- **수학 직관**: 반복 시행 → 빈도 → 확률 추정 (큰 수의 법칙 직관)
- **출처**: [06_확률1_분포_기댓값.md](../ML_수학_교과서/06_확률1_분포_기댓값.md)

#### 7. 거울 색칠 격자 (행렬/대칭)
- **룰**: 왼쪽 격자에 색칠된 패턴. 오른쪽은 비어 있음. 거울(좌우/상하/회전)을 골라 오른쪽을 똑같이 만들기
- **수학 직관**: 변환 = 함수. 같은 입력에 다른 변환 → 다른 결과
- **출처**: [02_선형대수2_행렬.md](../ML_수학_교과서/02_선형대수2_행렬.md)
