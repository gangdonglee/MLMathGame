# 3장. 선형대수 3 — 고유값, 대각화, SVD

> 행렬을 "어느 방향이 본질이고 그 방향이 얼마나 늘어나나" 로 분해하는 도구가 고유값과 SVD. ML 의 PCA, 화이트닝, NES 의 공분산 적응이 모두 이 분해 위에 서 있다.

---

## 3.0 한 장 요약

- 고유벡터 $v$: $Av = \lambda v$. 사상이 그 방향을 단순히 $\lambda$ 배 늘림.
- 대칭 행렬은 항상 **정규직교 고유벡터로 대각화** ($A = Q \Lambda Q^\top$).
- 일반 행렬은 SVD: $A = U \Sigma V^\top$. 입력측 ONB $V$, 출력측 ONB $U$, 양수 배율 $\Sigma$.
- PCA = 데이터 공분산의 고유분해. 가장 큰 고윳값 방향이 "분산이 가장 큰 방향".
- NES (10장) 의 공분산 적응은 SVD/대각화로 학습 분포를 회전·스케일.

---

## 3.1 고유값과 고유벡터

### 정의 3.1

$A \in \mathbb{R}^{n \times n}$, $v \in \mathbb{R}^n \setminus \{0\}$, $\lambda \in \mathbb{R}$ 가

$$
A v = \lambda v
$$

를 만족하면 $v$ 는 $A$ 의 **고유벡터**, $\lambda$ 는 대응 **고윳값**.

### 특성다항식

$Av = \lambda v \iff (A - \lambda I) v = 0$. $v \ne 0$ 이려면 $A - \lambda I$ 가 비가역, 곧

$$
\det(A - \lambda I) = 0.
$$

이 식은 $\lambda$ 에 대한 $n$ 차 다항식 (특성다항식). 실수해는 0 ~ $n$ 개.

### 예시 3.2

$A = \begin{pmatrix}2&1\\0&3\end{pmatrix}$. $\det(A - \lambda I) = (2-\lambda)(3-\lambda) = 0 \implies \lambda = 2, 3$. 대응 고유벡터 $(1, 0)^\top$, $(1, 1)^\top$.

### 직관

$Av = \lambda v$ 는 사상이 $v$ 의 *방향* 을 보존하고 *크기만* $\lambda$ 배. 모든 다른 방향은 일반적으로 회전 + 스케일이지만, 고유방향 위에서만 순수 스케일.

---

## 3.2 대각화

### 정의 3.3

$A \in \mathbb{R}^{n \times n}$ 가 **대각화 가능** = 가역 $P$ 와 대각 $D$ 가 존재해 $A = P D P^{-1}$.

이때 $P$ 의 열들은 $A$ 의 고유벡터, $D$ 의 대각 원소는 대응 고윳값.

### 정리 3.4 (n 개의 일차독립 고유벡터 ↔ 대각화 가능)

$A$ 가 일차독립인 $n$ 개의 고유벡터를 가지면, 그것들로 $P$ 를 구성하여 대각화 가능. 역도 성립. (증명 생략 — 표준 결과.)

### 거듭제곱과 함수 적용

대각화 $A = PDP^{-1}$ 가 있으면

$$
A^k = P D^k P^{-1}, \quad e^A = P e^D P^{-1}, \quad f(A) = P f(D) P^{-1}
$$

여기서 $f(D) = \text{diag}(f(\lambda_i))$. 이 성질이 미분방정식, Markov chain, Adam 의 분석 등에 깊이 쓰인다.

---

## 3.3 대칭 행렬의 스펙트럼 정리

### 정리 3.5 (실대칭행렬의 스펙트럼 정리)

$A \in \mathbb{R}^{n \times n}$ 가 대칭이면:

1. $A$ 의 모든 고윳값은 실수.
2. 서로 다른 고윳값에 대응하는 고유벡터는 직교.
3. ℝⁿ 은 $A$ 의 고유벡터로 이루어진 정규직교 기저를 가진다. 즉

$$
A = Q \Lambda Q^\top, \quad Q^\top Q = I, \quad \Lambda = \text{diag}(\lambda_1, \dots, \lambda_n).
$$

**증명 스케치 (1, 2).** $\lambda$ 가 복소 고윳값, $v$ 대응 복소 고유벡터라면 $\langle Av, v \rangle = \lambda \|v\|^2$ 이고 $\langle v, Av \rangle = \bar\lambda \|v\|^2$. 대칭성으로 두 값 같으므로 $\lambda = \bar\lambda$, 실수. 직교는 $\lambda_i \langle v_i, v_j \rangle = \langle Av_i, v_j \rangle = \langle v_i, A v_j \rangle = \lambda_j \langle v_i, v_j \rangle$, $\lambda_i \ne \lambda_j$ 이면 $\langle v_i, v_j \rangle = 0$. (3) 은 그람–슈미트 + 귀납. □

### 정리 3.6 (양정치 ↔ 모든 고윳값 양수)

대칭 $A$ 에 대해 $A$ 가 PD ↔ $\lambda_i > 0$ for all $i$. PSD ↔ $\lambda_i \ge 0$.

증명 스케치. $A = Q\Lambda Q^\top$ 에서 $x^\top A x = (Q^\top x)^\top \Lambda (Q^\top x) = \sum \lambda_i y_i^2$, $y = Q^\top x$. □

이 정리가 곧 PCA 와 가우스 분포의 음/양 판정 도구.

## 3.4 특이값 분해 (SVD)

### 정리 3.7 (SVD)

임의의 $A \in \mathbb{R}^{m \times n}$ 에 대해 다음 분해가 존재:

$$
A = U \Sigma V^\top
$$

- $U \in \mathbb{R}^{m \times m}$ 직교
- $V \in \mathbb{R}^{n \times n}$ 직교
- $\Sigma \in \mathbb{R}^{m \times n}$, 대각에 $\sigma_1 \ge \sigma_2 \ge \dots \ge \sigma_r > 0$ ($r = \text{rank}(A)$), 나머지 0.

$\sigma_i$ 를 **특이값(singular value)** 이라 부른다.

**증명 스케치.** $A^\top A$ 는 PSD 대칭. 정리 3.5 로 $A^\top A = V \Lambda V^\top$, $\Lambda = \text{diag}(\sigma_i^2)$. $u_i := \frac{1}{\sigma_i} A v_i$ ($\sigma_i > 0$ 인 경우) 가 정규직교임을 확인하고 $U$ 를 채움. 그러면 $A V = U \Sigma$, 양변 우측에 $V^\top$ 곱하면 결과. □

### 시선

SVD 는 임의의 사상을 다음 세 단계로 분해:

```
ℝⁿ ─[V^⊤ : 회전]─→ ℝⁿ ─[Σ : 좌표축 스케일]─→ ℝᵐ ─[U : 회전]─→ ℝᵐ
```

회전 → 스케일 → 회전. **모든 선형사상은 본질적으로 "축을 바꿔서 늘리는 것"**.

### 대칭과의 관계

$A$ 가 대칭이면 SVD 는 본질적으로 고유분해 (부호 흡수해서 $\sigma_i = |\lambda_i|$, $U = Q D$, $V = Q$).

## 3.5 랭크, 영공간, 열공간

### 정의 3.8

- 열공간 $\text{col}(A) = \{Ax : x \in \mathbb{R}^n\} \subseteq \mathbb{R}^m$.
- 영공간 $\ker A = \{x : Ax = 0\} \subseteq \mathbb{R}^n$.
- 랭크 $\text{rank}(A) = \dim \text{col}(A)$.

### 정리 3.9 (차원정리)

$\dim \ker A + \text{rank}(A) = n$.

**증명 스케치.** SVD 로 $A = U\Sigma V^\top$. $V$ 의 마지막 $n - r$ 열이 $\ker A$ 의 ONB, $U$ 의 처음 $r$ 열이 $\text{col}(A)$ 의 ONB. □

### ML 응용 — over-parameterization

PolicyNet 의 weight 1670 개는 "효과적인 차원" 보다 훨씬 많을 수 있다. 학습된 NN 의 layer weight 행렬을 SVD 해 보면 작은 특이값이 많고 큰 특이값은 소수 — *유효 랭크* 가 작다는 뜻. 이것이 모델 압축 / pruning 의 근거.

## 3.6 PCA — 분산이 가장 큰 방향

### 문제

데이터 $\{x_1, \dots, x_N\} \subset \mathbb{R}^d$ 가 주어졌을 때, 이를 가장 잘 보존하는 1차원 부분공간(정규벡터 $u$) 을 찾고 싶다. "잘 보존" 의 기준 = 사영의 분산 최대화.

### 풀이

$\bar x = \frac{1}{N}\sum x_i$, 중심화 $\tilde x_i = x_i - \bar x$. 공분산 (6장 정의)

$$
S = \frac{1}{N} \sum_i \tilde x_i \tilde x_i^\top \in \mathbb{R}^{d \times d}.
$$

$u$ 위의 사영 분산 = $u^\top S u$. 제약 $\|u\| = 1$ 에서 라그랑주:

$$
\nabla_u (u^\top S u - \lambda u^\top u) = 0 \implies S u = \lambda u.
$$

즉 **분산을 최대화하는 $u$ = $S$ 의 가장 큰 고윳값에 대응하는 고유벡터**, 분산 값 = $\lambda_{\max}$.

### 정리 3.10 (PCA 의 일반화)

$k \le d$ 차원 부분공간으로 사영해서 분산 합을 최대화 = $S$ 의 상위 $k$ 개 고유벡터를 골라 그것이 만드는 부분공간으로 사영. 이때 사영의 총 분산 = $\sum_{i=1}^k \lambda_i$.

증명은 Courant-Fischer 정리 (변분 표현) 사용. 직관: 직교 분해 후 큰 고윳값 방향부터 골라 가는 것.

## 3.7 화이트닝과 NES 미리보기

### 화이트닝

데이터 공분산이 $S$ 일 때 $S = Q \Lambda Q^\top$ 분해 후 변환 $z = \Lambda^{-1/2} Q^\top x$ 를 쓰면 $z$ 의 공분산이 $I$ 가 된다 (모든 방향이 같은 분산). NN 학습 안정화에 자주 쓰는 전처리.

### NES 의 공분산 적응 (preview)

표준 ES 는 $\epsilon \sim \mathcal{N}(0, I)$ — 모든 방향 동일 분산. NES (Natural Evolution Strategy) 는 분포를 $\mathcal{N}(\mu, C)$ 로 일반화하고 $C$ 도 같이 업데이트한다. 현 TinyCommand 의 [MLTrainer.cpp](../../ML/Tool/MLTrainer.cpp) 는 $C = \sigma^2 I$ 로 고정 — 가장 단순한 등방형. 만약 추후 NES 로 확장하면 $C$ 의 SVD 로 학습 분포를 회전 + 스케일하게 된다.

## 3.9 TinyCommand 연결

### 3.9.1 등방형 ES 는 SVD 가 자명

[MLTrainer.cpp](../../ML/Tool/MLTrainer.cpp) 가 사용하는 노이즈 분포는 $\mathcal{N}(0, \sigma^2 I)$. 공분산 $\sigma^2 I$ 의 SVD 는 $U = V = I$, $\Sigma = \sigma^2 I$. 즉 **회전 없음, 등방 스케일** — 가장 단순한 경우. 이 단순성은 코드 줄을 짧게 만들지만, "어떤 방향이 학습에 유리한가" 같은 정보는 *전혀 사용 안 함*. NES 가 필요해지는 동기.

### 3.9.2 PolicyNet 학습 후 weight SVD (가능한 분석)

`PolicyNet::SaveJson` 으로 저장된 1670 floats 를 다시 $W_1 \in \mathbb{R}^{32 \times 12}$, $W_2 \in \mathbb{R}^{32 \times 32}$, $W_3 \in \mathbb{R}^{6 \times 32}$ 로 풀어서 SVD 하면, 각 layer 의 *유효 랭크* 를 볼 수 있다. 보통 $W_2$ 가 32×32 이지만 효과적 랭크는 5~10 정도로 작은 경우가 많다 (over-parameterization). 가지치기/압축의 출발점.

### 3.9.3 PCA 로 본 feature 12 차원

`Policy::FeatureCount = 12` 의 12 차원 입력에 대해 PCA 를 적용하면, "가장 분산이 큰 feature 조합" 을 알 수 있다. 예: HP, 거리, 위협도가 강한 상관을 가지면 사실상 7~8 차원으로 줄어들지도 모름. 본 프로젝트는 직접 적용은 안 했지만, 만약 feature 가 폭발적으로 늘면 (예: 50+) PCA 또는 BatchNorm 으로 전처리하는 것이 표준.

## 3.10 연습문제

1. $A = \begin{pmatrix}3&1\\1&3\end{pmatrix}$ 의 고윳값/고유벡터를 구하고, $A = Q \Lambda Q^\top$ 분해를 적어라. PD 인지 확인하라.
2. $A \in \mathbb{R}^{3 \times 2}$, $A = \begin{pmatrix}1&0\\0&1\\1&1\end{pmatrix}$ 의 SVD 를 손으로 구하라. ($A^\top A$ 의 고유분해부터.)
3. 정리 3.6 (양정치 ↔ 고윳값 양수) 의 한 방향 ("PD ⇒ 모든 고윳값 양수") 을 직접 증명하라.
4. ℝ³ 데이터 $x_1 = (1,1,0), x_2 = (1,0,1), x_3 = (0,1,1)$ 의 평균과 공분산 $S$ 를 구하고 $S$ 의 고유분해로 PCA 의 첫 번째 주성분을 찾아라.
5. (TinyCommand) `PolicyNet` 의 $W_2$ 가 32×32 정사각이지만, ES 학습 후 SVD 했을 때 유효 랭크가 작을 가능성을 정리 3.9 와 차원정리로 설명하라. *유효 랭크가 작다는 것은 1670 차원의 weight 공간 중 실제로 사용되는 부분공간이 더 작다는 것을 의미한다.*
