# 7장. 확률 2 — 정규분포, CLT, MLE / MAP

> ML 의 *기본 노이즈 분포* 는 정규분포다. He 초기화, ES 노이즈, BatchNorm, dropout 의 평균/분산 분석. 이 장은 정규분포의 정의와 핵심 성질, 중심극한정리, 그리고 데이터로부터 모수를 추정하는 MLE / MAP 까지 다룬다.

---

## 7.0 한 장 요약

- 정규분포 $\mathcal{N}(\mu, \sigma^2)$ 는 평균/분산만으로 결정. 합/스케일에 닫혀 있음.
- 다변량 정규 $\mathcal{N}(\mu, \Sigma)$ 는 평균벡터 + 공분산행렬. 등방형 $\Sigma = \sigma^2 I$ 가 ES 의 기본 가정.
- **중심극한정리(CLT)**: iid 표본의 평균은 (적절히 정규화 후) 정규분포로 수렴. 정규분포가 *어디에서나* 나타나는 이유.
- **MLE** = 데이터의 likelihood 가 최대인 모수. NN 의 *대부분 손실 함수* (MSE, cross-entropy) 는 MLE 의 특수형.
- **MAP** = 사전분포를 곱한 posterior 의 최대. weight decay = Gaussian prior 의 MAP.

---

## 7.1 일변량 정규분포

### 정의 7.1

$X \sim \mathcal{N}(\mu, \sigma^2)$ ⟺ pdf

$$
f(x) = \frac{1}{\sqrt{2\pi}\, \sigma} \exp\left( -\frac{(x - \mu)^2}{2 \sigma^2} \right).
$$

$\mu \in \mathbb{R}$ 은 평균, $\sigma > 0$ 은 표준편차. 기댓값/분산 정리 (생략 — 가우스 적분).

### 표준화

$Z := (X - \mu)/\sigma \sim \mathcal{N}(0, 1)$ 이 표준 정규.

### 정리 7.2 (선형 변환과 합)

- $X \sim \mathcal{N}(\mu, \sigma^2) \implies aX + b \sim \mathcal{N}(a\mu + b, a^2 \sigma^2)$.
- $X_1 \sim \mathcal{N}(\mu_1, \sigma_1^2)$, $X_2 \sim \mathcal{N}(\mu_2, \sigma_2^2)$ 독립 ⇒ $X_1 + X_2 \sim \mathcal{N}(\mu_1 + \mu_2,\ \sigma_1^2 + \sigma_2^2)$.

증명은 적률생성함수 또는 푸리에 변환. ML 에서는 결과만 사용.

이 *닫힘성* 이 정규분포를 다루기 쉽게 만든다 (e.g., NN forward 의 가중합 역시 (입력이 정규면) 정규).

### 가우스 적분

$$
\int_{-\infty}^{\infty} e^{-x^2/2} dx = \sqrt{2\pi}.
$$

이 한 식이 정규분포의 정규화 상수 1/√(2π) 의 출처.

## 7.2 다변량 정규분포

### 정의 7.3

$X \in \mathbb{R}^d$ 가 $\mathcal{N}(\mu, \Sigma)$ ⟺ pdf

$$
f(x) = \frac{1}{(2\pi)^{d/2} (\det \Sigma)^{1/2}} \exp\left( -\frac{1}{2} (x - \mu)^\top \Sigma^{-1} (x - \mu) \right)
$$

($\Sigma$ 는 PD 대칭).

### 정리 7.4 (선형 변환)

$X \sim \mathcal{N}(\mu, \Sigma)$, $A \in \mathbb{R}^{m \times d}$, $b \in \mathbb{R}^m$ 이면

$$
AX + b \sim \mathcal{N}(A\mu + b,\ A \Sigma A^\top).
$$

특히 $\Sigma = LL^\top$ (Cholesky) 이고 $Z \sim \mathcal{N}(0, I)$ 이면 $X = LZ + \mu \sim \mathcal{N}(\mu, \Sigma)$. 이것이 일반 정규벡터를 *표준 정규 + 선형변환* 으로 시뮬하는 표준 트릭.

### 등방형

$\Sigma = \sigma^2 I$ 인 경우 모든 좌표가 독립 표준정규 × $\sigma$. ES 의 노이즈가 이 형태.

### 화이트닝 (3장 §3.7 다시)

$Y = \Sigma^{-1/2}(X - \mu)$ 로 변환하면 $Y \sim \mathcal{N}(0, I)$. 학습 안정화 / 데이터 전처리의 표준 도구.

## 7.3 중심극한정리 (CLT)

### 정리 7.5 (CLT)

iid $X_1, X_2, \dots$ 가 $\mu = E[X]$, $\sigma^2 = \text{Var}(X) < \infty$ 면

$$
\frac{\bar X_n - \mu}{\sigma / \sqrt n} \overset{d}{\to} \mathcal{N}(0, 1) \quad (n \to \infty),
$$

$\overset{d}{\to}$ 는 분포 수렴.

**증명 스케치.** 적률생성함수 $M_{(\bar X_n - \mu)\sqrt n / \sigma}(t)$ 를 테일러 전개하면 $e^{t^2/2}$ 로 수렴. □

### 직관

평균은 결국 정규분포 모양. 이것이 "왜 자연/측정/금융 데이터가 종 모양인가" 의 부분 답.

### ML 에서의 의미

- **He 초기화**: 한 뉴런으로 들어오는 가중합 $\sum w_j x_j$ 는 (적당한 조건 하에) 근사적으로 정규. 분산을 잘 통제하면 forward 신호가 폭발/소멸 안 함.
- **미니배치 그래디언트**: $B$ 개의 샘플로 추정한 그래디언트는 $\mathcal{N}$ 근사. 학습률 / 배치 크기 trade-off 의 이론적 도구.
- **드롭아웃의 기댓값 정규화**: 드롭아웃 후 평균 출력의 분산이 CLT 로 예측 가능.

## 7.4 likelihood 와 MLE

### 정의 7.6 (likelihood)

데이터 $\mathcal{D} = \{x_1, \dots, x_n\}$ 와 모수 $\theta$ 의 모델 $p(x | \theta)$ 가 주어졌을 때

$$
\mathcal{L}(\theta) := p(\mathcal{D} | \theta) = \prod_{i=1}^n p(x_i | \theta).
$$

### 정의 7.7 (MLE)

$$
\hat\theta_{\text{MLE}} := \arg\max_\theta \mathcal{L}(\theta) = \arg\max_\theta \sum_i \log p(x_i | \theta).
$$

(곱 → log 합으로 바꾸면 수치 안정 + 미분 쉬움.)

### 정리 7.8 (정규분포의 MLE)

$x_1, \dots, x_n \overset{iid}{\sim} \mathcal{N}(\mu, \sigma^2)$ 의 MLE:

$$
\hat\mu = \bar x = \tfrac{1}{n} \sum x_i, \quad \hat\sigma^2 = \tfrac{1}{n} \sum (x_i - \bar x)^2.
$$

**증명.** Log-likelihood

$$
\ell(\mu, \sigma^2) = -\tfrac{n}{2} \log(2\pi \sigma^2) - \tfrac{1}{2\sigma^2} \sum (x_i - \mu)^2.
$$

$\partial \ell / \partial \mu = 0 \implies \mu = \bar x$. $\partial \ell / \partial \sigma^2 = 0 \implies \sigma^2 = \tfrac{1}{n} \sum (x_i - \bar x)^2$. □

### MSE = 정규 likelihood

회귀 모델 $y = f(x; \theta) + \epsilon$, $\epsilon \sim \mathcal{N}(0, \sigma^2)$ 가정. Log-likelihood

$$
\ell(\theta) = -\frac{1}{2\sigma^2} \sum (y_i - f(x_i; \theta))^2 + \text{const}.
$$

$\theta$ 에 대한 최대화 ⟺ $\sum (y_i - f)^2$ 의 최소화 = **MSE**.

즉 MSE 손실은 *"노이즈가 가우시안" 가정의 MLE*. 다른 분포 가정 (예: Laplace) 이면 다른 손실 (L1).

### Cross-entropy = 카테고리 likelihood (예고)

분류 문제 ($y \in \{1, \dots, K\}$, 모델 $p_\theta(y | x)$). Log-likelihood = $\sum \log p_\theta(y_i | x_i)$ = (-) cross-entropy. 8장 정보이론에서 직접 다룬다.

### MLE 의 일관성 (정리만)

규칙적인 조건 하에 $\hat\theta_{\text{MLE}} \overset{P}{\to} \theta^*$ (진짜 모수). 또한 $\sqrt n (\hat\theta - \theta^*) \overset{d}{\to} \mathcal{N}(0, I^{-1})$, $I$ = Fisher 정보. 이 점근정규성이 신뢰구간 / 가설검정의 근거.

## 7.5 베이즈와 MAP

### 정의 7.9 (사후분포)

사전 $p(\theta)$, likelihood $p(\mathcal{D} | \theta)$. 베이즈 정리

$$
p(\theta | \mathcal{D}) = \frac{p(\mathcal{D} | \theta) p(\theta)}{p(\mathcal{D})} \propto p(\mathcal{D} | \theta) p(\theta).
$$

### 정의 7.10 (MAP)

$$
\hat\theta_{\text{MAP}} := \arg\max_\theta p(\theta | \mathcal{D}) = \arg\max_\theta \big[\log p(\mathcal{D} | \theta) + \log p(\theta)\big].
$$

MLE 와 차이는 $\log p(\theta)$ 의 추가.

### 가우시안 prior = L2 정규화

$\theta \sim \mathcal{N}(0, \tau^2 I)$ 사전이면 $\log p(\theta) = -\tfrac{1}{2\tau^2} \|\theta\|^2 + \text{const}$. MAP:

$$
\hat\theta = \arg\min_\theta \left[ -\log p(\mathcal{D} | \theta) + \tfrac{1}{2\tau^2} \|\theta\|^2 \right].
$$

이것이 **weight decay (L2 regularization)**. 두 번째 항은 weight 가 커지는 것에 페널티.

### 라플라스 prior = L1

$\theta_j$ 가 독립 라플라스 ⇒ $\log p(\theta) \propto -\|\theta\|_1$. MAP 가 LASSO. 희소한 weight 를 만든다.

## 7.6 He 초기화 — 정규분포의 응용

### 분석

NN 의 한 뉴런: $z = \sum_{j=1}^{n} w_j x_j$. weight $w_j \overset{iid}{\sim} \mathcal{N}(0, \sigma_w^2)$, 입력 $x_j$ 는 ReLU 통과 후 평균 0 분산 $\sigma_x^2$ (가정).

$\text{Var}(z) = \sum \text{Var}(w_j x_j) = n \sigma_w^2 \sigma_x^2$.

ReLU 통과 후 분산은 절반 (음수 영역이 0): $\sigma_{\text{ReLU}(z)}^2 = \tfrac{1}{2} \text{Var}(z) = \tfrac{n}{2} \sigma_w^2 \sigma_x^2$.

**조건**: layer 통과해도 분산이 보존 ($\sigma_{\text{ReLU}(z)}^2 = \sigma_x^2$):

$$
\sigma_w^2 = \frac{2}{n} \implies \sigma_w = \sqrt{\frac{2}{n}}.
$$

이것이 **He 초기화** (정의: 정규 $\mathcal{N}(0, 2/n_{\text{in}})$). $n_{\text{in}}$ = fan-in.

### TinyCommand 의 적용

PolicyNet 의 layer 별 fan-in:

| Layer | fan-in | $\sigma_w$ |
|---|---|---|
| L1 | 12 | $\sqrt{2/12} \approx 0.408$ |
| L2 | 32 | $\sqrt{2/32} \approx 0.250$ |
| L3 | 32 | $\sqrt{2/32} \approx 0.250$ |

[PolicyNet.cpp](../../ML/AI/PolicyNet.cpp) 의 `RandomInit` 가 정확히 이 식. `scale` 파라미터로 전체 배수 조절 (작게 하면 초기 점수가 평탄해서 ES 가 다양한 argmax 를 보게 됨).

## 7.8 TinyCommand 연결

### 7.8.1 ES 노이즈 = 등방 정규벡터

[MLTrainer.cpp](../../ML/Tool/MLTrainer.cpp) 의 `std::normal_distribution<float>(0, 1)` 출력 1670 개 = $\epsilon \sim \mathcal{N}(0, I_{1670})$. 이를 $\sigma$ 배 → $\sigma\epsilon \sim \mathcal{N}(0, \sigma^2 I)$ (정리 7.4).

### 7.8.2 He 초기화 = 정리 7.6 의 응용

위 §7.6 그대로. CLT 가 "fan-in 이 충분히 크면 $z$ 가 근사적으로 정규" 를 보장하므로 분산 통제만으로 forward 가 안정.

### 7.8.3 weight decay 가 없는 이유

ES 학습은 GD 가 아니므로 손실에 $\|\theta\|^2$ 정규화를 더하는 식의 "MAP" 표현이 자연스럽지 않다. 대신 *암묵적 정규화* 가 작동:

- $\sigma$ 가 학습률 역할도 겸함 → 노이즈 자체가 weight 를 제한된 영역에 머물게.
- $J$ 의 평균값이 발산하는 영역 (예: NN 출력이 너무 큼) 은 점수가 떨어져 그쪽으로 안 감.

NN 의 weight 폭발이 관측되면 명시적 weight decay 도입 가능 ($\theta \leftarrow (1 - \beta) \theta + \alpha \hat g$ 형태).

### 7.8.4 시뮬 점수의 분포

같은 $\theta$ 로 *다른 맵* 을 평가한 점수들의 분포는 일반적으로 정규 아님 (이산 사건 + 시간 적분). 그러나 *맵 풀 평균* 은 CLT 의 영향으로 점점 정규에 가까워짐. 맵 풀이 충분히 크면 한 generation 의 J 추정이 근사적으로 정규.

### 7.8.5 winner archive 의 분포 사용

`winnersDir` 에 dump 된 weight 들은 학습 중 우수한 후보의 분포. 이 분포의 평균 / 공분산을 분석하면 *weight 공간의 좋은 영역의 모양* 을 추정 가능 — NES 로 가는 자연스러운 다음 단계.

## 7.9 연습문제

1. $X \sim \mathcal{N}(\mu, \sigma^2)$ 의 평균 / 분산을 정의 7.1 의 적분으로 직접 유도하라 (가우스 적분 사용).
2. 두 독립 표준정규 $X, Y$ 에 대해 $X + 2Y$ 의 분포를 구하라.
3. CLT 의 한 사례: $X_i \sim \text{Bernoulli}(p)$, $S_n = \sum X_i$. $(S_n - np) / \sqrt{np(1-p)} \to \mathcal{N}(0, 1)$ 을 정리 7.5 로부터 유도하라.
4. 베르누이 데이터 $x_1, \dots, x_n \in \{0, 1\}$, $p \in (0,1)$ 의 MLE $\hat p$ 를 직접 구하라.
5. $\theta \sim \mathcal{N}(0, \tau^2 I)$ 사전 + iid $y_i = \theta^\top x_i + \epsilon_i$, $\epsilon_i \sim \mathcal{N}(0, \sigma^2)$ likelihood 의 MAP 추정 $\hat\theta$ 의 닫힌 형식 (ridge 회귀) 을 유도하라.
6. (TinyCommand) `PolicyNet::RandomInit(seed, scale=0.5)` 가 학습 시작 단계의 NN 출력 분산에 어떻게 영향을 주는지 §7.6 의 분석으로 정량 설명하라. ES 의 다양성 (argmax 가 자주 바뀜) 에 어떤 효과?
