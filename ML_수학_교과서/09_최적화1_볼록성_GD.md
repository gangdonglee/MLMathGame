# 9장. 최적화 1 — 볼록성, Gradient Descent

> "더 좋은 weight 로 가는 절차" 의 형식 정의가 *최적화*. 이 장은 볼록 / 매끄러움의 정의, GD 의 한 step 이 무엇을 보장하는지, 그리고 SGD 가 왜 작동하는지 정리한다.

---

## 9.0 한 장 요약

- **볼록집합**: 두 점을 잇는 선분이 집합 안에 있음. **볼록함수**: epigraph 가 볼록.
- 미분가능 볼록함수: 1차 부등식 $f(y) \ge f(x) + \nabla f(x)^\top (y - x)$.
- $L$-매끄러움 (그래디언트 립시츠): GD 의 한 step 보장 $f(x_{t+1}) \le f(x_t) - \tfrac{\alpha}{2} \|\nabla f(x_t)\|^2$ ($\alpha \le 1/L$).
- GD 수렴: 볼록 + 매끄러움 ⇒ $f(x_t) - f^* = O(1/t)$. Strongly convex 면 지수 수렴.
- SGD: 미니배치 그래디언트 = 진짜 그래디언트의 *불편 추정*. 학습률 다이어트 $\alpha_t \to 0$ 로 수렴.
- NN 손실은 *비볼록* — 보장은 약하지만 실제로 잘 작동. 안장점 / 평탄 영역 / 초기화의 역할.

## 9.1 볼록성

### 정의 9.1 (볼록집합)

$C \subseteq \mathbb{R}^n$ 이 **볼록** ⟺ $\forall x, y \in C, \lambda \in [0, 1]$,

$$
\lambda x + (1 - \lambda) y \in C.
$$

### 정의 9.2 (볼록함수)

$f : C \to \mathbb{R}$ ($C$ 볼록) 가 **볼록** ⟺

$$
f(\lambda x + (1-\lambda) y) \le \lambda f(x) + (1-\lambda) f(y).
$$

엄격한 부등식 (등호는 $x = y$ 만) 이면 **strictly convex**.

### 예시 9.3

- 일변수: $x^2$, $e^x$, $-\log x$, $|x|$ (strictly 는 아님)
- $\mathbb{R}^n$: 모든 노름, $\tfrac{1}{2} x^\top A x$ ($A$ PSD), affine $a^\top x + b$
- 합성: 볼록 + 비감소-볼록 합성은 볼록 ($g \circ f$ 에서 $g$ 비감소-볼록, $f$ 볼록)
- 최대값: $\max_i f_i$ ($f_i$ 볼록) 는 볼록

### 정리 9.4 (1차 특성)

$f$ 가 미분가능 + 볼록 ⟺

$$
f(y) \ge f(x) + \nabla f(x)^\top (y - x) \quad \forall x, y.
$$

**증명 스케치 (⇒).** 볼록 정의에 $z = \lambda y + (1-\lambda) x$ 대입, $\lambda \to 0^+$ 극한. 양변을 $\lambda$ 로 나눠 정리하면 우변 = 좌변. □

이 부등식이 *볼록함수의 가장 자주 쓰이는 형태*. 그래디언트가 *접평면이 함수 아래에 있음* 을 보장.

### 정리 9.5 (2차 특성)

$f \in C^2$ 이면 $f$ 볼록 ⟺ 헤시안 $H(x) \succeq 0$ (PSD) for all $x$. **Strictly convex** ⟺ $H \succ 0$ (PD).

증명. 정리 9.4 의 미적분 결과. □

### Strong convexity

$f$ 가 $\mu$-강볼록 ($\mu > 0$) ⟺

$$
f(y) \ge f(x) + \nabla f(x)^\top (y - x) + \tfrac{\mu}{2} \|y - x\|^2.
$$

동치: $H \succeq \mu I$. 더 강한 곡률 보장.

### 정리 9.6 (볼록의 좋은 성질)

- 볼록함수의 local min 은 global min.
- Strictly convex 면 minimizer 가 존재하면 유일.

이 성질들이 볼록 최적화를 *해결 가능* 으로 만든다. NN 의 손실은 비볼록 → 이 보장 없음.

## 9.2 매끄러움

### 정의 9.7 ($L$-매끄러움)

$f$ 가 $L$-smooth ⟺ $\nabla f$ 가 $L$-립시츠:

$$
\|\nabla f(x) - \nabla f(y)\| \le L \|x - y\|.
$$

동치 (2차 미분 가능): $\|H(x)\| \le L$ for all $x$.

### 정리 9.8 (descent lemma)

$f$ 가 $L$-smooth 이면

$$
f(y) \le f(x) + \nabla f(x)^\top (y - x) + \tfrac{L}{2} \|y - x\|^2.
$$

**증명.** 일변수 $\phi(t) = f(x + t(y - x))$ 의 미분 $\phi'(t) = \nabla f(x + t(y-x))^\top (y - x)$. $\phi(1) - \phi(0) = \int_0^1 \phi' dt$. $\phi'(t) - \phi'(0) = (\nabla f(x + t(y-x)) - \nabla f(x))^\top (y-x) \le L t \|y-x\|^2$ (CS). 적분 ⇒ $f(y) - f(x) - \nabla f(x)^\top (y-x) \le \tfrac{L}{2}\|y-x\|^2$. □

### 의미

descent lemma 는 "1차근사에서 떨어지는 양은 거리² × L/2 이내" 를 보장. GD 의 step 분석의 출발점.

## 9.3 Gradient Descent

### Update

$$
x_{t+1} = x_t - \alpha \nabla f(x_t).
$$

### 정리 9.9 (한 step 보장 — descent)

$f$ 가 $L$-smooth, $\alpha \in (0, 1/L]$. 그러면

$$
f(x_{t+1}) \le f(x_t) - \tfrac{\alpha}{2} \|\nabla f(x_t)\|^2.
$$

**증명.** Descent lemma 에 $y = x_{t+1} = x_t - \alpha g$, $g = \nabla f(x_t)$:

$$
f(x_{t+1}) \le f(x_t) - \alpha \|g\|^2 + \tfrac{L \alpha^2}{2} \|g\|^2 = f(x_t) - \alpha (1 - \tfrac{L\alpha}{2}) \|g\|^2.
$$

$\alpha \le 1/L$ 이면 $1 - L\alpha/2 \ge 1/2$. □

이 정리는 *볼록성 없이도* 작동. 즉 NN (비볼록) 에서도 학습률만 작으면 매 step 손실이 단조감소.

### 따름정리 9.10 (그래디언트 노름의 합)

위 가정에서 $T$ step 후

$$
\sum_{t=0}^{T-1} \|\nabla f(x_t)\|^2 \le \frac{2 (f(x_0) - f^*)}{\alpha}.
$$

⇒ $\min_{t < T} \|\nabla f(x_t)\| = O(1/\sqrt T)$. *임계점 근방* 으로 수렴 (global min 이라는 보장은 비볼록에서는 없음).

### 정리 9.11 (볼록 + 매끄러움 — 함숫값 수렴)

$f$ 볼록 + $L$-smooth, $\alpha = 1/L$ 이면

$$
f(x_T) - f^* \le \frac{2 L \|x_0 - x^*\|^2}{T}.
$$

$O(1/T)$ 율. 증명은 정리 9.9 의 합산 + 1차 부등식.

### 정리 9.12 (강볼록 — 지수 수렴)

$f$ 가 $\mu$-강볼록 + $L$-smooth, $\alpha = 1/L$, $\kappa := L/\mu$. 그러면

$$
f(x_T) - f^* \le (1 - 1/\kappa)^T (f(x_0) - f^*).
$$

지수 수렴. $\kappa$ = 조건수가 작을수록 빠름. 데이터 정규화 / 화이트닝의 동기.

## 9.4 SGD

### 동기

NN 손실 $L(\theta) = \tfrac{1}{N} \sum_{i=1}^N \ell(\theta; x_i, y_i)$. $\nabla L$ 한 번 계산이 *전체 데이터* 패스 필요 → 느림. 미니배치 $B \subset \{1, \dots, N\}$ 로 추정:

$$
\hat g = \tfrac{1}{|B|} \sum_{i \in B} \nabla \ell(\theta; x_i, y_i).
$$

### 정리 9.13 (불편성)

$B$ 가 균등 무작위 추출이면 $E[\hat g] = \nabla L(\theta)$. □

### Update

$$
\theta_{t+1} = \theta_t - \alpha_t \hat g_t.
$$

### 정리 9.14 (Robbins–Monro 수렴 조건)

학습률 시퀀스 $\{\alpha_t\}$ 가

$$
\sum_t \alpha_t = \infty, \quad \sum_t \alpha_t^2 < \infty
$$

를 만족하면 (그리고 노이즈 분산 유한, 볼록성 등의 표준 조건) SGD 가 $f^*$ 에 a.s. 수렴.

예: $\alpha_t = c/t$. 또는 $c/\sqrt t$.

### 실제 ML 에서

위 조건의 $\alpha_t$ 는 너무 빨리 감소 ⇒ 진척 느림. 실용에서는 *상수 학습률* + 가끔 step decay + warmup. 수렴은 *근방까지* 만 보장되지만 충분 (NN 은 어차피 정확한 minimum 이 의미 없음 — overfitting).

## 9.5 비볼록 최적화 — NN 의 현실

### 손실 풍경

NN 의 $\mathcal L(\theta)$ 는 비볼록. 그러나 다음 경험적 사실들:

1. **고차원에서 local min ≈ global min**. 다양한 초기화에서 수렴한 솔루션의 손실이 비슷.
2. **임계점은 거의 안장점** (5장 §5.4). PD 헤시안의 임계점은 드뭄.
3. **평탄한 minimum 이 generalization 에 좋다**. SGD 의 노이즈가 평탄한 영역 선호.

### "왜 SGD 가 비볼록에서 잘 되는가" 의 부분 답

- 노이즈가 안장점 탈출에 도움.
- batch 크기와 학습률의 trade-off → 효과적인 *온도* 가 학습 후반 줄어들면서 점점 좁은 영역 탐색.
- Implicit regularization (L2 와 비슷한 효과를 SGD 자체가 만든다는 가설).

증명은 활발한 연구 영역. 본 교재는 직관 수준으로 둠.

### NN 학습 안정 도구

- **Batch normalization**: layer 입력의 분산을 1 로 강제. 효과적으로 손실 풍경의 조건수 $\kappa$ 를 낮춤.
- **Skip connections (ResNet)**: 그래디언트가 깊은 layer 까지 잘 전달. vanishing gradient 회피.
- **학습률 스케줄**: warmup + cosine decay 가 표준.

## 9.7 ES 와 GD 의 관계

### ES = "확장된 GD"

ES 의 update $\theta \leftarrow \theta + \alpha \hat g$ 은 GD 의 update 와 같은 형태. 차이는 $\hat g$ 의 *원천*:

| | GD | SGD | ES |
|---|---|---|---|
| $g$ | $\nabla L$ 정확 | 미니배치 ∇ | $J_\sigma$ 의 표본 ∇ |
| 비용 | 1 forward + 1 backward | $|B|$ forward + back | $N$ forward |
| 노이즈 | 0 | 데이터 샘플링 | 노이즈 + 데이터 |
| 미분 가정 | 필요 | 필요 | 불필요 |

GD/SGD 의 수렴 정리 (정리 9.11) 가 ES 에 직접 적용되지 않지만, 부드러운 $J_\sigma$ 에 대한 수렴 분석이 가능 (Nesterov-Spokoiny 등).

### 정리 9.15 (ES 의 수렴 — 형식적)

$J_\sigma$ 가 매끄럽고 적절한 학습률이면 ES 의 $E[\theta_t]$ 가 $J_\sigma$ 의 임계점으로 수렴.

(자세한 증명은 NES 논문들. ML 교과서 §1.3 의 직관 + 9.3 의 GD 분석을 표본 평균으로 일반화.)

## 9.8 TinyCommand 연결

### 9.8.1 ES update = GD 식

[MLTrainer.cpp](../../ML/Tool/MLTrainer.cpp) 의 한 줄 `theta += learningRate * g_hat` 은 정리 9.9 의 GD update. learning rate 0.05 는 정확한 $L$ 을 모르고 정한 보수적인 값.

### 9.8.2 학습률 안정성

만약 한 generation 후 `bestFit` 이 떨어지면 $\alpha$ 가 너무 큰 가능성. 정리 9.9 의 $\alpha < 1/L$ 위반. 해결: $\alpha$ 를 줄이거나 $\sigma$ 를 줄임 ($J_\sigma$ 의 매끄러움 $L$ 이 $1/\sigma$ 정도에 비례 — 더 부드러우면 안전).

### 9.8.3 비볼록 J_σ

$J(\theta)$ 는 시뮬 결과 (이산 + 비볼록). $J_\sigma$ 는 매끄럽지만 *여전히 비볼록*. 따라서 ES 의 수렴 보장도 *임계점까지* 만. 학습 로그에서 plateau 가 보이면 안장점일 가능성.

### 9.8.4 Map pool = 데이터 분포

`mapPool.size()` 가 sample size $N$. 각 평가의 *맵 풀 평균* 이 $J(\theta)$ 의 표본추정 (정리 9.13 의 SGD 와 같은 구조). 맵 풀이 작으면 $J$ 추정 노이즈 大 → ES gradient 노이즈 大 → 학습 불안정.

### 9.8.5 early stop

`earlyStopFitness = F` 옵션은 $J_\sigma(\theta_t) > F$ 시 중단. 정리 9.10 의 "임계점 근방 도달" 의 약한 형태 — 손실값 기준이지 그래디언트 기준은 아님. NN 학습의 표준 관행.

## 9.9 연습문제

1. $f(x, y) = x^2 + 4 y^2$ 가 볼록임을 정의 9.2 와 정리 9.5 두 방법으로 보여라. $L, \mu$ 를 구하라.
2. 정리 9.4 의 1차 부등식에서, 양변에 $\nabla f(y)^\top (x - y)$ 를 더하는 트릭으로 $\langle \nabla f(x) - \nabla f(y), x - y \rangle \ge 0$ (단조성) 을 유도하라.
3. $f(x) = \tfrac{1}{2} \|x\|^2$ 에 GD ($\alpha = 1$) 를 1 step 적용하면 정확히 $f^* = 0$ 에 도달함을 보여라. 일반 $\alpha$ 에서는?
4. 정리 9.11 의 증명을 정리 9.4 + 정리 9.9 의 합산으로 직접 채워라.
5. (TinyCommand) `MLTrainer::Params { sigma = 0.10, learningRate = 0.05 }` 의 조합에서 만약 $J$ 가 비교적 매끄럽다고 가정하고 $L \approx 1/\sigma = 10$ 으로 추정한다면, 정리 9.9 의 안전한 $\alpha$ 범위는 어디인가? 현재 값은 그 범위 안인가?
