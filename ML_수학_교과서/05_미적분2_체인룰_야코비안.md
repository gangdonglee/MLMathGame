# 5장. 미적분 2 — 체인룰, 야코비안, 헤시안

> 4장의 그래디언트는 "스칼라 함수 → 벡터" 였다. NN 은 *벡터 → 벡터 → 벡터* 의 합성이므로 야코비안 행렬과 다변수 체인룰이 필요. 이 장의 결과가 곧 backpropagation 의 수학적 정당성.

---

## 5.0 한 장 요약

- $f : \mathbb{R}^n \to \mathbb{R}^m$ 의 미분은 야코비안 $J_f \in \mathbb{R}^{m \times n}$ — 각 출력 좌표를 각 입력 좌표로 편미분한 표.
- 체인룰: $J_{g \circ f}(x) = J_g(f(x)) \cdot J_f(x)$. 두 행렬의 곱.
- 스칼라 손실 $L : \mathbb{R}^m \to \mathbb{R}$ 의 그래디언트를 입력으로 끌어내릴 때 $\nabla_x L = J_f^\top \nabla_y L$. **전치** 가 핵심.
- 헤시안 $H = \nabla^2 f$ 는 대칭 행렬. 임계점 분류, 곡률 정보, Newton 법.
- Backpropagation = layer 마다 야코비안 전치를 오른쪽에서 곱해 나가는 절차. 동적 프로그래밍.

---

## 5.1 벡터값 함수와 야코비안

### 정의 5.1 (야코비안)

$f : U \subseteq \mathbb{R}^n \to \mathbb{R}^m$, $f = (f_1, \dots, f_m)^\top$ 가 모든 편미분이 존재할 때

$$
J_f(a) := \begin{pmatrix}
\frac{\partial f_1}{\partial x_1} & \cdots & \frac{\partial f_1}{\partial x_n} \\
\vdots & \ddots & \vdots \\
\frac{\partial f_m}{\partial x_1} & \cdots & \frac{\partial f_m}{\partial x_n}
\end{pmatrix}(a) \in \mathbb{R}^{m \times n}.
$$

### 시선

- 행: 각 $f_i$ 의 그래디언트 $\nabla f_i^\top$.
- 사상: $J_f(a)$ 는 $f$ 의 $a$ 근처 *최선의 선형 근사*.

### 정의 5.2 (미분가능 — 일반)

$f$ 가 $a$ 에서 미분가능 = $\exists L \in \mathbb{R}^{m \times n}$ s.t. $f(a+h) = f(a) + L h + o(\|h\|)$. 이때 $L = J_f(a)$.

### 예시 5.3

$f(x_1, x_2) = (x_1 x_2,\ x_1 + x_2^2,\ \sin x_1)$. 야코비안

$$
J_f = \begin{pmatrix} x_2 & x_1 \\ 1 & 2 x_2 \\ \cos x_1 & 0 \end{pmatrix} \in \mathbb{R}^{3 \times 2}.
$$

## 5.2 체인룰

### 정리 5.4 (다변수 체인룰)

$f : \mathbb{R}^n \to \mathbb{R}^m$, $g : \mathbb{R}^m \to \mathbb{R}^p$, $f$ 가 $a$, $g$ 가 $f(a)$ 에서 미분가능이면 $g \circ f$ 도 $a$ 에서 미분가능이고

$$
J_{g \circ f}(a) = J_g(f(a)) \cdot J_f(a) \in \mathbb{R}^{p \times n}.
$$

**증명 스케치.** 1차근사 $f(a+h) = f(a) + J_f h + o(\|h\|)$ 와 $g(y + k) = g(y) + J_g k + o(\|k\|)$ 를 결합. $k = J_f h + o(\|h\|)$ 로 두면 $g(f(a+h)) = g(f(a)) + J_g (J_f h) + o(\|h\|)$. □

### 시선 — 행렬곱

체인룰의 핵심은 "도함수가 행렬이 되니까 합성의 도함수도 *행렬곱* 이 된다". 1변수의 $f'(g(x)) g'(x)$ 의 자연스러운 일반화.

### 따름정리 5.5 (스칼라 손실의 끌어내리기)

$L : \mathbb{R}^m \to \mathbb{R}$ (스칼라), $y = f(x)$ 일 때

$$
\nabla_x L = J_f^\top \nabla_y L.
$$

**증명.** $L \circ f$ 의 야코비안은 1×n 행벡터 $\nabla_x L^\top$. 정리 5.4 에서 $\nabla_x L^\top = \nabla_y L^\top \cdot J_f$. 양변 전치하면 결과. □

이 식이 *백프롭의 수학적 본질*. 각 layer 의 야코비안의 *전치* 를 곱하면서 출력측 그래디언트를 입력측 그래디언트로 옮긴다.

## 5.3 합성으로 본 NN forward / backward

### Forward (정의)

$L$ 개 layer 의 NN. $h^{(0)} = x$, layer $\ell$:

$$
z^{(\ell)} = W^{(\ell)} h^{(\ell-1)} + b^{(\ell)}, \quad h^{(\ell)} = \phi(z^{(\ell)}).
$$

손실 $\mathcal{L} = L(h^{(L)}, y)$.

### Backward — 체인룰의 응용

각 layer 에서 **출력측 그래디언트** $\delta^{(\ell)} := \partial \mathcal{L} / \partial z^{(\ell)}$ 를 정의. 그러면

1. (활성화 통과) $\partial \mathcal{L} / \partial h^{(\ell)} = (\phi'(z^{(\ell)}) \odot)^{-1}$ ... 정확히는 $\delta^{(\ell)} = \frac{\partial \mathcal L}{\partial h^{(\ell)}} \odot \phi'(z^{(\ell)})$. ($\odot$ = 원소별 곱)
2. (선형층 통과) $\frac{\partial \mathcal L}{\partial h^{(\ell-1)}} = (W^{(\ell)})^\top \delta^{(\ell)}$. ← 따름정리 5.5
3. (weight 의 그래디언트) $\frac{\partial \mathcal L}{\partial W^{(\ell)}} = \delta^{(\ell)} (h^{(\ell-1)})^\top$, $\frac{\partial \mathcal L}{\partial b^{(\ell)}} = \delta^{(\ell)}$.

### 직관 — 동적 프로그래밍

$\delta^{(L)}$ 부터 거꾸로 $\delta^{(L-1)}, \dots, \delta^{(1)}$ 을 계산하면, 각 step 이 *행렬-벡터 곱 한 번*. 모든 weight 의 그래디언트를 forward 와 같은 시간 (≈$O(\sum \dim W^{(\ell)})$) 에 얻는다. Naive 방식이라면 weight 마다 별도 forward 를 다시 돌려야 해 $\Omega(\#W \cdot \text{forward})$.

### ReLU 의 미분

$\phi(z) = \max(0, z)$. 미분은 $\phi'(z) = \mathbb{1}[z > 0]$ (0 에서는 정의 X — subgradient 0 또는 1 으로 둠). 이산이지만 거의 모든 점에서 잘 정의.

### 죽은 ReLU 와 그래디언트 0

$z < 0$ 인 뉴런은 $\phi'(z) = 0$ ⇒ $\delta = 0$ ⇒ weight 그래디언트 $= 0$ ⇒ SGD update 0 ⇒ 영영 같은 위치. He 초기화 (7장) 가 입력 분포의 절반쯤만 음수가 되도록 균형을 잡아 이 문제를 완화.

## 5.4 헤시안

### 정의 5.6 (헤시안)

$f : \mathbb{R}^n \to \mathbb{R}$, 두 번 미분 가능. 헤시안은

$$
H_f(a) := \nabla^2 f(a) := \begin{pmatrix}
\frac{\partial^2 f}{\partial x_1^2} & \cdots & \frac{\partial^2 f}{\partial x_1 \partial x_n} \\
\vdots & \ddots & \vdots \\
\frac{\partial^2 f}{\partial x_n \partial x_1} & \cdots & \frac{\partial^2 f}{\partial x_n^2}
\end{pmatrix}(a) \in \mathbb{R}^{n \times n}.
$$

### 정리 5.7 (Schwarz / Clairaut)

$f$ 의 모든 2계 편미분이 연속이면 $\frac{\partial^2 f}{\partial x_i \partial x_j} = \frac{\partial^2 f}{\partial x_j \partial x_i}$. 즉 $H$ 는 대칭.

### 다변수 테일러

$$
f(a + h) = f(a) + \nabla f(a)^\top h + \tfrac{1}{2} h^\top H_f(a) h + o(\|h\|^2).
$$

### 정리 5.8 (임계점의 2차 분류)

$\nabla f(a) = 0$ 이라 하자.

- $H(a)$ PD ⇒ $a$ 는 strict local min
- $H(a)$ ND (모든 고윳값 음수) ⇒ strict local max
- $H(a)$ 의 고윳값이 양수와 음수를 동시에 가짐 ⇒ 안장점

증명. 테일러로 충분히 작은 $h$ 에 대해 $f(a+h) - f(a) \approx \tfrac{1}{2} h^\top H h$. PD 면 $h^\top H h > 0$ for all $h \ne 0$. □

### NN 손실의 안장점

NN 의 손실은 *비볼록* 이고, **고차원에서 임계점은 거의 항상 안장점** (random matrix 이론). local min 이 우려보다 적은 이유. ES 와 SGD 가 안장점에서 빠져나가는 능력이 다르다 — ES 는 노이즈 분포 자체가 사방으로 퍼져 있어 자연스럽게 빠져나옴.

## 5.5 야코비안의 응용 모음

### 5.5.1 Newton 방법

$\nabla f(\theta) = 0$ 을 풀려고 $\nabla f$ 를 1차근사:

$$
\nabla f(\theta + \Delta) \approx \nabla f(\theta) + H(\theta) \Delta = 0 \implies \Delta = -H^{-1} \nabla f.
$$

수렴이 빠르지만 $H^{-1}$ 계산 비용 ($O(n^3)$) 이 NN 에서는 비현실적. → quasi-Newton (BFGS, L-BFGS), 또는 1차 근사 (Adam 의 분산 보정) 로 우회.

### 5.5.2 라그랑주 승수

제약 $g(x) = 0$ 아래 $f(x)$ 의 극값:

$$
\nabla f = \lambda \nabla g.
$$

PCA (3장), SVM, weight 정규화 등에 사용.

### 5.5.3 변환의 부피 변화

$f$ 가 미분가능 가역, $J_f$ 가 그 야코비안. 변수 변환 적분

$$
\int_{f(D)} \phi(y) dy = \int_D \phi(f(x)) \, |\det J_f(x)| \, dx.
$$

정규분포의 변환 (7장 §화이트닝), 확률밀도 변환에서 기본.

## 5.6 ES 가 ∇를 안 쓰는 이유 — 다시

4장 §4.8 에서 ES 는 *부드러운* $J_\sigma$ 의 ∇를 추정한다고 했다. 백프롭은 *원래* $J$ 의 ∇를 정확히 계산하지만:

| | ES | Backprop |
|---|---|---|
| 필요한 것 | $J$ 만 (forward 평가) | $J$ 가 미분 가능 ($J_f$) |
| 호출 횟수 | $N$ 번 평가 | 1 번 forward + 1 번 backward |
| 노이즈 | 큼 (분산 큼) | 0 (정확) |
| 비미분 손실 | OK | X — subgradient 트릭 필요 |
| 환경 시뮬 | OK | X — 시뮬 자체가 미분 불가 |

**TinyCommand 의 J(θ) = 시뮬 한 판 점수** 는 (a) 시뮬 내부 if/else 분기 (b) 충돌, 사망 같은 이산 사건 (c) 시간 적분 등으로 *미분 불가능*. 따라서 백프롭 적용 불가, ES 가 자연스러운 선택. NN 의 forward 가 미분 가능해도 그 *상위* 가 미분 불가면 백프롭은 끊긴다.

## 5.8 TinyCommand 연결

### 5.8.1 PolicyNet 의 forward = 합성

[PolicyNet.cpp](../../ML/AI/PolicyNet.cpp) 의 `Forward` 는 정확히 §5.3 의 합성 패턴. 만약 backprop 을 구현한다면:

- $\delta^{(3)} = \nabla_{\text{score}} \mathcal{L}$ (마지막 layer 활성화 없으므로 그대로)
- $\delta^{(2)} = (W_3^\top \delta^{(3)}) \odot \mathbb{1}[z^{(2)} > 0]$
- $\delta^{(1)} = (W_2^\top \delta^{(2)}) \odot \mathbb{1}[z^{(1)} > 0]$
- 각 weight 그래디언트 = $\delta^{(\ell)} (h^{(\ell-1)})^\top$

### 5.8.2 ES 와 backprop 혼합 가능성

만약 *교사 데이터* (예: 사람이 labeling 한 "이 상황 → 이 행동") 가 있으면, NN 을 cross-entropy 손실로 supervised pretraining → 그 후 ES 로 fine-tune 하는 hybrid 가 가능. supervised 단계는 정리 5.4–5.5 의 백프롭을 그대로 사용. 본 프로젝트는 현재 단계 미적용.

### 5.8.3 ES 의 노이즈 = 사실상 finite-difference 그래디언트

식 $\hat g = \frac{1}{N\sigma}\sum J(\theta + \sigma\epsilon_i)\epsilon_i$ 는 $N$ 개의 sided finite-difference 의 가중평균과 본질적으로 동치. antithetic 사용 시 *중심 차분* (central difference) 의 일반화:

$$
\hat g = \frac{1}{N\sigma}\sum \tfrac{J(\theta + \sigma\epsilon_i) - J(\theta - \sigma\epsilon_i)}{2} \cdot 2\epsilon_i.
$$

(2 가 상쇄되어 1장 §1.4 의 antithetic 식과 동일.) 즉 ES 는 야코비안의 finite-diff 추정 + 표본평균.

### 5.8.4 "안장점" 과 1670 차원

PolicyNet 의 1670 차원 손실 풍경에서 임계점은 정리 5.8 으로 분류. 고차원이라 *완전 PD* 인 임계점은 거의 없음 (random matrix 이론: 모든 고윳값이 양수일 확률 ≈ exp(−n)). 따라서 학습이 정체되는 평탄한 영역은 보통 *안장점* 의 근방. ES 가 안장점에서 빠져나오는 능력이 SGD 보다 강한 경향 (노이즈 σ 가 모든 방향).

4. 2-layer MLP $\hat y = W_2 \, \text{ReLU}(W_1 x)$, 손실 $\mathcal L = \tfrac{1}{2} \|y - \hat y\|^2$ 에 대해 $\partial \mathcal L / \partial W_1$, $\partial \mathcal L / \partial W_2$ 를 §5.3 의 절차로 직접 유도하라.
