# 1장. 선형대수 1 — 벡터, 내적, 노름

> ML 의 모든 입력 / weight / gradient 는 결국 **실수 벡터** 다. PolicyNet 의 1670개 weight 도 ℝ^1670 의 한 점, ES 가 만드는 노이즈 ε 도 같은 공간의 한 점. 이 장은 그 "공간" 의 정의와, 거리 / 각도를 재는 도구인 내적과 노름을 정리한다.

---

## 1.0 한 장 요약

- 벡터는 "방향과 크기를 가진 화살표" 이자 동시에 "숫자 리스트". ℝⁿ 은 둘을 같은 것으로 보는 공간.
- **내적** $\langle x, y \rangle = \sum x_i y_i$ 는 두 벡터가 얼마나 같은 방향을 보는지 잰다.
- **노름** $\|x\| = \sqrt{\langle x, x \rangle}$ 는 벡터의 크기. 코시–슈바르츠 부등식 $|\langle x, y \rangle| \le \|x\| \|y\|$ 가 모든 후속 정리의 뿌리.
- 정규직교(orthonormal) 기저는 좌표 계산을 가장 단순하게 한다. SVD / He 초기화 / antithetic 노이즈가 모두 이 개념에 의존.

---

## 1.1 벡터 공간 ℝⁿ

### 정의 1.1 (실벡터 공간)

집합 $V$ 와 두 연산 (덧셈 $+ : V \times V \to V$, 스칼라곱 $\cdot : \mathbb{R} \times V \to V$) 이 다음을 만족하면 $V$ 를 **실벡터 공간** 이라 한다.

1. $(V, +)$ 는 가환군 (덧셈에 대해 결합/교환/항등원 0/역원).
2. $\alpha (\beta x) = (\alpha \beta) x$, $1 \cdot x = x$.
3. 분배: $\alpha (x + y) = \alpha x + \alpha y$, $(\alpha + \beta) x = \alpha x + \beta x$.

본 교재가 거의 항상 다루는 예는 다음 하나다.

$$
\mathbb{R}^n = \{ (x_1, x_2, \dots, x_n) \mid x_i \in \mathbb{R} \}
$$

원소 하나를 **벡터** 라 부르고, $x = (x_1, \dots, x_n)^\top$ (열벡터) 로 적는다.

### 두 가지 시선

| 시선 | 설명 | 쓰임 |
|---|---|---|
| 기하 | "원점에서 뻗은 화살표" | 그래디언트 방향, 내적의 cos θ 해석 |
| 대수 | "$n$ 개의 숫자 리스트" | 코드의 `float w[N]`, JSON 의 배열 |

ML 에서는 두 시선을 자유롭게 오간다. 예를 들어 PolicyNet 의 weight 는 코드에서 `float w[1670]` (대수) 이지만, ES 가 "어느 방향으로 한 발 옮겨야 하나" 를 묻는 순간 같은 객체를 ℝ^1670 안의 화살표(기하) 로 본다.

## 1.2 부분공간, 일차결합, 기저

### 정의 1.2 (부분공간)

$W \subseteq V$ 가 다음 두 조건을 만족하면 부분공간:

1. $0 \in W$
2. $x, y \in W$ 와 $\alpha, \beta \in \mathbb{R}$ 에 대해 $\alpha x + \beta y \in W$

### 정의 1.3 (일차결합 / 일차독립)

벡터 $v_1, \dots, v_k$ 의 **일차결합** 은 $\alpha_1 v_1 + \dots + \alpha_k v_k$ 의 모든 식. 만약

$$
\alpha_1 v_1 + \dots + \alpha_k v_k = 0 \implies \alpha_1 = \dots = \alpha_k = 0
$$

이면 $\{v_i\}$ 는 **일차독립**.

### 정의 1.4 (기저, 차원)

$V$ 의 일차독립인 부분집합 $B$ 가 $V$ 를 생성 ($\text{span}(B) = V$) 하면 $B$ 는 **기저**. 모든 기저는 같은 원소 개수를 가지며, 그것을 $V$ 의 **차원** $\dim V$ 라 한다.

ℝⁿ 의 표준 기저는 $e_i = (0, \dots, 0, \underbrace{1}_{i\text{번째}}, 0, \dots, 0)^\top$. 따라서 $\dim \mathbb{R}^n = n$.

> **예시 1.5** PolicyNet 의 weight 공간은 $\mathbb{R}^{1670}$. 표준 기저는 "$i$ 번째 weight 만 1, 나머지 0". ES 의 노이즈 $\epsilon \in \mathbb{R}^{1670}$ 는 이 표준 기저의 일차결합이며, 1670 개의 독립적인 진동 방향을 동시에 흔든다.

## 1.3 내적

### 정의 1.6 (유클리드 내적)

$x, y \in \mathbb{R}^n$ 에 대해

$$
\langle x, y \rangle := \sum_{i=1}^n x_i y_i = x^\top y
$$

### 정리 1.7 (내적의 성질)

1. (대칭) $\langle x, y \rangle = \langle y, x \rangle$
2. (선형) $\langle \alpha x + \beta y, z \rangle = \alpha \langle x, z \rangle + \beta \langle y, z \rangle$
3. (양정치) $\langle x, x \rangle \ge 0$, 등호는 $x = 0$ 일 때만

증명. 정의로부터 직접 확인. □

### 직관 — 같은 방향을 보는 정도

$\langle x, y \rangle$ 가 크면 두 벡터가 같은 방향, 0 이면 직각, 음수면 반대 방향. 코사인 법칙으로 정확히 적으면

$$
\langle x, y \rangle = \|x\| \|y\| \cos \theta
$$

여기서 $\theta$ 는 두 벡터 사이 각도.

> **TinyCommand 미리보기** ES 의 gradient 추정 $\hat{g} = \frac{1}{N\sigma} \sum r_i \epsilon_i$ 를 보면, 각 항은 "노이즈 방향 $\epsilon_i$ 를 점수의 부호 $r_i$ 만큼 가중" 한 것이다. 결과적으로 $\hat{g}$ 와 임의의 $\epsilon_i$ 의 내적 $\langle \hat{g}, \epsilon_i \rangle$ 가 $r_i$ 와 양의 상관을 가진다. 즉 ES 는 **내적이 큰 방향을 새 θ 로 잡는다**.

## 1.4 노름

### 정의 1.8 (노름)

함수 $\|\cdot\| : V \to \mathbb{R}_{\ge 0}$ 가 다음을 만족하면 노름:

1. $\|x\| = 0 \iff x = 0$
2. $\|\alpha x\| = |\alpha| \|x\|$
3. $\|x + y\| \le \|x\| + \|y\|$ (삼각부등식)

내적이 있는 공간에서는 **유도 노름** $\|x\| := \sqrt{\langle x, x \rangle}$ 가 자연스럽다. 이를 $\ell^2$ 또는 유클리드 노름이라 부른다.

### 다른 노름 — 비교

| 노름 | 식 | 비고 |
|---|---|---|
| $\ell^1$ | $\|x\|_1 = \sum |x_i|$ | LASSO 정규화 |
| $\ell^2$ | $\|x\|_2 = \sqrt{\sum x_i^2}$ | 유클리드, 가장 흔함 |
| $\ell^\infty$ | $\|x\|_\infty = \max_i |x_i|$ | 최악값 보호 |

본 교재는 별 표기 없으면 $\ell^2$.

### 정리 1.9 (코시–슈바르츠 부등식)

$\mathbb{R}^n$ 의 모든 $x, y$ 에 대해

$$
|\langle x, y \rangle| \le \|x\| \, \|y\|
$$

등호는 $y = \alpha x$ 인 경우만.

**증명.** $y = 0$ 이면 자명. 아니면 임의의 $\lambda \in \mathbb{R}$ 에 대해

$$
0 \le \|x - \lambda y\|^2 = \|x\|^2 - 2 \lambda \langle x, y \rangle + \lambda^2 \|y\|^2.
$$

오른쪽은 $\lambda$ 에 대한 이차식, 항상 ≥ 0 이려면 판별식 ≤ 0:

$$
4 \langle x, y \rangle^2 - 4 \|x\|^2 \|y\|^2 \le 0 \implies \langle x, y \rangle^2 \le \|x\|^2 \|y\|^2. \qquad \square
$$

이 부등식이 왜 중요한가: cos θ 를 정의하려면 $-1 \le \frac{\langle x, y \rangle}{\|x\|\|y\|} \le 1$ 이 보장돼야 하는데, 그 보장이 곧 코시–슈바르츠다.

### 따름정리 1.10 (삼각부등식)

$\|x + y\|^2 = \|x\|^2 + 2 \langle x, y \rangle + \|y\|^2 \le \|x\|^2 + 2 \|x\| \|y\| + \|y\|^2 = (\|x\| + \|y\|)^2$. □

## 1.5 직교와 정규직교 기저

### 정의 1.11

- $\langle x, y \rangle = 0$ 이면 $x \perp y$ (직교).
- 기저 $\{u_1, \dots, u_n\}$ 가 $\langle u_i, u_j \rangle = \delta_{ij}$ (i = j 면 1, 아니면 0) 를 만족하면 **정규직교 기저** (orthonormal basis, ONB).

### 정리 1.12 (정규직교 기저에서의 좌표)

$\{u_i\}$ 가 ℝⁿ 의 ONB 면, 임의의 $x \in \mathbb{R}^n$ 에 대해

$$
x = \sum_{i=1}^n \langle x, u_i \rangle \, u_i, \qquad \|x\|^2 = \sum_{i=1}^n \langle x, u_i \rangle^2.
$$

**증명.** $x = \sum \alpha_i u_i$ 로 두고 $\langle \cdot, u_j \rangle$ 양변 내적을 취하면 $\langle x, u_j \rangle = \alpha_j$. 두 번째는 $\|x\|^2 = \langle x, x \rangle = \sum_i \alpha_i^2$ 로부터. □

이 정리가 SVD (3장) 와 PCA 의 출발점이다. 또한 He 초기화 (7장) 가 정규분포의 좌표별 독립성을 가정해 weight 를 채우는 근거이기도 하다.

### 그람–슈미트 직교화 (스케치)

임의의 일차독립 $\{v_1, \dots, v_n\}$ 에서 ONB $\{u_1, \dots, u_n\}$ 를 만드는 절차:

$$
u_1 = \frac{v_1}{\|v_1\|}, \quad
\tilde u_k = v_k - \sum_{j=1}^{k-1} \langle v_k, u_j \rangle u_j, \quad
u_k = \frac{\tilde u_k}{\|\tilde u_k\|}.
$$

**핵심 직관:** 각 $v_k$ 에서 이미 만든 방향 성분을 빼고 남은 부분만 정규화. 결과적으로 매번 새 $u_k$ 는 이전 모든 $u_j$ 와 직교.

## 1.6 거리, 각도, 사영

### 거리

두 점 사이 거리 $d(x, y) := \|x - y\|$. ML 에서 중요한 예:

- **MSE** (평균 제곱 오차): $\frac{1}{N} \sum (y_i - \hat y_i)^2 = \frac{1}{N} \|y - \hat y\|^2$
- **유사도**: 흔히 $1 - \cos\theta(x, y)$

### 사영

벡터 $x$ 의 단위벡터 $u$ 위로의 **정사영**:

$$
\text{proj}_u x = \langle x, u \rangle \, u
$$

이 값은 $x$ 의 $u$ 방향 성분. 나머지 $x - \text{proj}_u x$ 는 $u$ 와 직교.

### 정리 1.13 (피타고라스)

$x \perp y$ 이면 $\|x + y\|^2 = \|x\|^2 + \|y\|^2$.

**증명.** $\|x+y\|^2 = \|x\|^2 + 2 \langle x, y \rangle + \|y\|^2 = \|x\|^2 + \|y\|^2$. □

| 개념 | 어디서 다시 나오나 |
|---|---|
| ℝⁿ 벡터 | 모든 weight, feature, gradient |
| 내적 $x^\top y$ | 선형 정책의 score = $\langle w, \text{feature} \rangle$ |
| 노름 $\|\cdot\|$ | weight decay, gradient clipping |

| 정규직교 기저 | SVD (3장), 표준 노이즈 $\epsilon \sim \mathcal{N}(0, I)$ (7장) |

## 1.8 TinyCommand 연결

### 1.8.1 feature 벡터 ([PolicyFeatures.h](../../ML/AI/PolicyFeatures.h))

`Policy::FeatureCount = 12`. 매 tick 에 `Compute(...)` 가 12 개 float 을 채워 ℝ¹² 의 한 점 $\phi(\text{state})$ 를 만든다. 정의 1.1 의 ℝⁿ 의 가장 작은 실제 사례.

### 1.8.2 선형 score = 내적

[PolicyWeights.cpp](../../ML/AI/PolicyWeights.cpp) 의 `Score(action, features)`:

```
score(a) = Σ_i w[a][i] * feature[i] = ⟨ w_a, φ ⟩
```

여기서 $w_a \in \mathbb{R}^{12}$, $\phi \in \mathbb{R}^{12}$. 정확히 정의 1.6 의 내적.

### 1.8.3 ES 의 노이즈 = ℝ^N 의 점

[MLTrainer.cpp](../../ML/Tool/MLTrainer.cpp) 가 `std::normal_distribution<float>` 으로 1670 개 float 을 뽑으면, 그것은 ℝ^1670 의 한 점. antithetic 으로 두 점 $\epsilon, -\epsilon$ 을 같이 만드는 것은 "원점 대칭인 두 화살표".

### 1.8.4 He 초기화 = 노름 스케일

[PolicyNet.cpp](../../ML/AI/PolicyNet.cpp) 의 `RandomInit` 에서 L1 의 weight 를 $\sigma = \sqrt{2/12}$ 로 뽑는 이유는, 한 뉴런으로 들어오는 12 개 입력의 가중합 $\sum w_i x_i$ 의 노름을 적당히 통제해 ReLU 가 "절반은 살아있고 절반은 죽도록" 만들기 위함. 1.4 의 노름 + 7장의 정규분포가 만나는 자리.
