# ML을 위한 수학 교과서 — TinyCommand 사례 연계

이 문서는 [Docs/ML_교과서/](../ML_교과서/) 가 풀어 쓴 OpenAI ES + NN 정책의 **밑단 수학** 을 따로 정리한 교재이다. 자매 교재가 "코드와 직관" 에 무게를 둔다면, 이 교재는 그 직관 뒤의 **정의 — 정리 — (간단한) 증명** 을 담당한다.

> **이 교재의 수준**
>
> - 출발: 고등학교 미적분 / 행렬 / 확률 정도
> - 도착: 학부 1~3학년. 편미분, 행렬곱, 고윳값, 정규분포의 MLE, KL divergence, gradient descent 의 수렴 정리까지.
> - 주요 정리는 **증명을 적되 직관 먼저, 형식 나중** 의 순서. 증명이 길거나 측도론이 필요한 부분은 *증명 스케치* 로 대체.
> - 모든 개념은 가능한 한 [TinyCommand](https://github.com/gangdonglee/Tiny-Command) 의 ES/NN 코드 한 줄과 연결한다 (각 장 마지막 §x.N).

## 읽는 방법

- 각 장 도입부의 **§x.0 한 장 요약** 만 먼저 훑어 흐름을 잡고, 필요한 절만 깊게 읽어도 된다.
- "**정의**" 박스는 형식 정의. "**정리**" 박스는 증명 (또는 스케치) 동반. "**예시**" 는 ℝ² 같은 작은 차원의 손계산.
- "**TinyCommand 연결**" 절(각 장 끝)은 그 장의 수학이 실제 [ML/](../../ML/) 코드 어느 줄에서 쓰이는지 보여 준다.
- LaTeX 수식 사용. GitHub / VS Code preview 에서 렌더링됨. 인라인 `$...$`, 디스플레이 `$$...$$`.

## 표기 규약

| 기호 | 의미 |
|---|---|
| $\mathbb{R}$, $\mathbb{R}^n$ | 실수 집합, $n$ 차원 실수 벡터 공간 |
| $x, y, \theta, \epsilon$ | (보통) 벡터. 굵은 표시 안 함. 문맥으로 구분 |
| $A, B, J, H$ | 행렬 |
| $\langle x, y \rangle$ 또는 $x^\top y$ | 내적 |
| $\|x\|$, $\|x\|_2$ | 노름 (기본 = 유클리드) |
| $\nabla f(\theta)$ | $f$ 의 그래디언트 (열벡터로 둠) |
| $J_f$ | $f$ 의 야코비안 행렬 |
| $E[X]$, $\text{Var}(X)$ | 기댓값, 분산 |
| $X \sim \mathcal{N}(\mu, \sigma^2)$ | $X$ 가 정규분포를 따름 |
| $H(p)$, $D_{KL}(p \| q)$ | 엔트로피, KL divergence |
| $\theta \leftarrow \theta + \alpha g$ | 대입 (코드 `=` 와 동일) |

> **ML 의 수학** = (선형대수: weight 의 표현) + (미적분: gradient 의 정의) + (확률: 노이즈와 기댓값) + (정보이론: 손실의 의미) + (최적화: 더 나은 weight 로 가는 절차).
